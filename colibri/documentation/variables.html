Variables

  1. Introduction
  2. Defining variables

Introduction

Opis Colibri allows developers to define and store custom values for later usage. Variables are collected using the Variables method. 

use Opis\Colibri\Define;

Define::Variables(function($variables){

    // Collect variables

});

You can override a variable by specifying a lower collecting priority. 

use Opis\Colibri\Define;

Define::Variables(function($variables){

    // Collect variables

}, -1);

The collector passed to the callback function is an object that implements the Opis\Colibri\VariableCollectorInterface interface. 

interface VariableCollectorInterface
{
    /**
     * Register a new variable
     *
     * @param   string  $name       Variable's name
     * @param   mixed   $value      Variable's value
     *
     * @return  \Opis\Colibri\VariableCollectorInterface   Self reference
     */
    
    public function register($name, $value);
    
    /**
     * Register multiple variables at once
     *
     * @param   array   $variables  An array of variables that will be registered
     *
     * @return  \Opis\Colibri\VariableCollectorInterface   Self reference
     */
    
    public function bulkRegister(array $variables);
}

Defining variables

Defining a new variable is done by passing the variable's name and value to the register method. 

use Opis\Colibri\Define;

Define::Variables(function($variables){

    // Register a variable named `foo`
    $variables->register('foo', 2015);

});

You can register multiple variables at once by using the bulkRegister method. The key of the array passed as argument to the bulkRegister method represents the variable's name and the value represents the variable's actual value. 

use Opis\Colibri\Define;

Define::Variables(function($variables){

    // Register a variable named `foo`
    $variables->bulkRegister(array(
        'foo' => 2015,
        'bar' => 'baz',
    ));

});

Using variables

Variables can be used with the help of the Variable global function. This function takes as an argument the name of the variable you want to retrieve. 

$value = Variable('foo');
