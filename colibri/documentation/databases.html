Database connections

  1. Introduction
  2. Defining database connections
  3. Using database connections

Introduction

Opis Colibri provides a database abstraction layer with the help of the Opis Database library. Database connections are collected using the Connections method. 

use Opis\Colibri\Define;

Define::Connections(function($connection){

    // Collect database connections

});

You can override a database connection by specifying a lower collecting priority. 

use Opis\Colibri\Define;

Define::Connections(function($connection){

    // Collect database connections

}, -1);

The collector passed to the callback function is an object that implements the Opis\Colibri\ConnectionCollectorInterface interface. 

interface ConnectionCollectorInterface
{
    /**
     * Defines a new database connection
     *
     * @param   string      $name       Connection name
     * @param   \Closure    $callback   Connection constructor callback
     * @param   boolean     $default    (optional) Default flag
     *
     * @return  \Opis\Colibri\ConnectionCollectorInterface  Self reference
     */
    
    public function create($name, Closure $callback, $default = false);
}

Defining database connections

Database connections are defined by passing a connection name and a callback closure to the create method. The callback closure must return an instance of the Opis\Database\Connection class. Optionally, you can pass true, as the third argument of the create method, if you want to mark the connection as being the default connection. If a database connection was defined when the framework was installed, then that database connection will be used as the default database connection and it cannot be overridden. 

use Opis\Colibri\Define;
use Opis\Database\Connection;

Define::Connections(function($connection){

    $connection->create('MyConnection', function(){
       return new Connection('mysql:localhost;dbname=somedb', 'user', 'pass'); 
    });
    
    $connection->create('DefaultConnection', function(){
       return new Connection('mysql:localhost;dbname=otherdb', 'user', 'pass'); 
    }, true);

});

Using database connections

Database connections are used in conjunction with the Database and Schema global functions, by passing the connection's name, as an argument, to the functions. 

    
$items = Database('MyConnection')->from('foo')
                                 ->select()
                                 ->all();

Schema('MyConnection')->create('bar', function($table){
   // ...
});

If the connection's name is omitted, then the default database connection will be used. 

    
$items = Database()->from('foo')
                   ->select()
                   ->all();

Schema()->create('bar', function($table){
   // ...
});
