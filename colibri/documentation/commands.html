Commands

  1. Introduction
  2. Registering new commands

Introduction

Beside the default commands provided by the framework, users can define their own commands. The console command system is based on the Symfony Console component. All user defined commands are collected using the Commands method. 

use Opis\Colibri\Define;

Define::Commands(function($command){

    // Collect commands

});

You can override commands by specifying a lower collecting priority. 

use Opis\Colibri\Define;

Define::Commands(function($command){

    // Collect commands

}, -1);

The collector passed to the callback function is an object that implements the Opis\Colibri\CommandCollectorInterface interface. 

interface CommandCollectorInterface
{
    /**
     * Register a new command
     *
     * @param   string      $name       Command's name
     * @param   callable    $callback   Callback
     *
     * @return  \Opis\Colibri\CommandCollectorInterface   Self reference
     */
    
    public function register($name, $callback);
}

Registering new commands

Commands can be registered by passing the command's name and a callback to the register method. The callback passed to the register method must return an instance of a class that inherits from Symfony\Component\Console\Command\Command. You will find here all the documentation you need in order to start creating your own commands. 

use Opis\Colibri\Define;

Define::Commands(function($command){

    $command->register('foo', function(){
        return new FooCommand();
    });

});
