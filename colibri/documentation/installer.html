Introduction

Each module can optionally have an associated installer. To be more precise, a module's installer is just a class that extends the abstract Opis\Colibri\ModuleInstaller class. The class must be declared under the Opis\Colibri\ModuleInstaller namespace, by using an installer file.

The name of the class it's given by the module's name. So if you make an installer for a module named foo the name of the class will be Foo. If you make an installer for a module named foo-bar the name of the class will be FooBar. 

namespace Opis\Colibri\ModuleInstaller;

use Opis\Colibri\ModuleInstaller;

class Example extends ModuleInstaller
{

}

The installer file

The installer file is where module's installer must be declared. If this file is missing, then the module doesn't have an associated installer. By default, the file's name is formed from the name of the module followed by the .install.php suffix. You can change the name of the installer file by adding an installer property into the module.json file.

Usage

The installer class is an important part of a module. It allows a module, for example, to create new tables into a database, when the module gets installed, and to remove them when the module is uninstalled.

The installer class has 4 methods that can be overridden: install, enable, disable and uninstall. Each method corresponds to an operation that can be performed over a module and is executed when that operation takes place. 

namespace Opis\Colibri\ModuleInstaller;

use Opis\Colibri\ModuleInstaller;

class Example extends ModuleInstaller
{
    
    public function install()
    {
        //executed when the module is installed
    }
    
    public function uninstall()
    {
        //executed when the module is uninstalled
    }
    
    public function enable()
    {
        //executed when the module is enabled
    }
    
    public function disable()
    {
        //executed when the module is disabled
    }
}

If you want to see a usage example of an installer, take a look at user module's installer class.
