Assets

  1. Introduction

Introduction

Modules can carry their own CSS files, JavaScript files, images, videos or other kind of web assets as long they are all gathered into a single folder.

Making those assets publicly available is simply a matter of adding an assets property into the module.json file. The property must contains the assets folder's name.

Important! A module's assets can be accesed only if that module is enabled.

Assuming that your module is named foo and you have an asset named bar, you can access your asset at /assets/module/foo/bar.
